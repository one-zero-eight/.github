## Our projects

The projects we are working on are open-source, and we are always looking for contributors.

The core project is [InNoHassle ecosystem](https://innohassle.ru). It includes several projects:

- [Website](https://github.com/one-zero-eight/InNoHassle-Website) - main website of the ecosystem which provides access to all the services.
- [Schedule backend (Events)](https://github.com/one-zero-eight/InNoHassle-Events) - API for the schedule of classes, sports, cleaning etc.
- [Music room backend](https://github.com/one-zero-eight/InNoHassle-MusicRoom) - API for the Music room booking.
- [Music room bot](https://github.com/one-zero-eight/InNoHassle-MusicRoomBot) - bot for the Music room booking which is the main user interface for the Music room API.

Work in progress:

- [InNoHassle Bot](https://github.com/one-zero-eight/InNoHassle-Bot) - bot for the Schedule service and some other services of the ecosystem.
- [Accounts](https://github.com/one-zero-eight/InNoHassle-Accounts) - service for managing the user accounts, authentication and authorization.
- [Rooms backend](https://github.com/one-zero-eight/InNoHassle-Rooms) - API for managing the tasks in your dormitory room.
- [Rooms bot](https://github.com/one-zero-eight/InNoHassle-RoomsBot) - bot for the Rooms API.
